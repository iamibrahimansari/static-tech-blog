Write a program in C programming language to find/calculate average of given elements of integer array?

An average is a single number taken as representative of a list of numbers, usually the sum of the numbers divided by how many numbers are in the list. For example, the average of the numbers 2, 3, 4, 7, and 9 is 5 because (2 + 3 + 4 + 7 + 9) / 5 = 5

You are given a size and elements of unsigned integer array which means non negative integer element. Your job is to calculate average of elemnts of given array. Size and elemnts must recieve by the user and make sure that the size of array given by the user must be natural number which means greater than 0 otherwise print "Array can't be constrcut with size 0." if user enter negative size then print "Array size can't be negative." Elements of array also must be whole numbers or non-negative integers otherwise simply print "Sorry, negative element is not allow." on screen.

Example 1

Input

Enter the size of given array: 5

Enter 5 elements for given array: 1 2 3 4 5

Output

Average of elements of given array: 3

Explanation

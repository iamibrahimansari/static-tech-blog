Write a program in C programming language to calculate sum of digits of given number?

You are given a whole number which means greater than or equal to 0. Your job is to calculate sum of all the digits of given number. If user enter negative number then print "Your input is invalid." on screen.

Example 1

Input

Enter a whole number(>=0): 100

Output

Sum of digits of given number 100 is 1.

Explanation

The given number 100 which has 3 digits 1, 0 and 0. After add these digits we get 1 because 1 + 0 + 0 = 1.

Example 2

Input

Enter a whole number(>=0): 123

Output

Sum of digits of given number 123 is 6.

Explanation

The given number 1234 which has 4 digits 1, 2, 3 and 4. After add these digits we get 10 because 1 + 2 + 3 + 4 = 10.

Example 3

Input

Enter a whole number(>=0): -2

Output

Your input is invalid.

Explanation

According to the question when user enter negative number then must be print "Your input is invalid." on screen.

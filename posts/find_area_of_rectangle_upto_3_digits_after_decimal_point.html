Write a program in C programming language to find / calculate area of given rectangle up-to 3 digits after decimal point, if area contains non zero digit otherwise convert it into an integer number?

The area of a rectangle depends on its sides. Basically, the formula for area is equal to the product of length and length of the breadth. Area of rectangle = length x breadth or we can say that Area of rectangle = width x height

You are given width and height of rectangle. Your job is to find area of given rectangle. If area of rectangle doesn't contain any non-zero digits after decimal point(ex: 13.000000) then print "13" otherwise print area up to 3 digits after decimal point. Area can't be negative that's why if user enter 0 or negative values for either width or height then print "Your input is invalid." on screen.

Example 1

Input

Enter width and height of given rectangle: 5 2

Output

Area of given rectangle is 10

Explanation

Area of Rectangle = width x height. Here width is 5 but it converts into 5.000000 and height is 2 and it converts into 2.000000 that's why 5.000000 x 2.000000 = 10.000000 but again we have converted it into an integer number that's why the result is 10.

Example 2

Input

Enter width and height of given rectangle: 5.2 2.5

Output

Area of given rectangle is 13

Explanation

Area of Rectangle = width x height. Here width is 5.2 but it converts into 5.200000 and height is 2.5 and it converts into 2.500000 that's why 5.200000 x 2.500000 = 13.000000 but again we have converted it into an integer number that's why the result is 13.

Example 3

Input

Enter width and height of given rectangle: 5.2 2

Output

Area of given rectangle is 10.400

Explanation

Area of Rectangle = width x height. Here width is 5.2 but it converts into 5.200000 and height is 2 and it converts into 2.000000 that's why 5.200000 x 2.000000 = 10.400000 but according to the question, we have to print only 3 digits after decimal point if the result contains non zero values after decimal point that's why the result is 10.400

Example 4

Input

Enter width and height of given rectangle: 5.2 0

Output

Your input is invalid.

Explanation

Side of rectangle can't be zero that's why input is invalid.

Write a program in C programming language to check whether a given natural number is prime or not?

A prime number is a number that has exactly two factors, 1 and the number itself. For example 3 is a prime number because 3 only divisible by 1 and itself which means it has only 2 factors.

Your job is to first take a natural number from the user to check whether it is a prime number or not. If the given number is a natural number then print as per the test cases which is shown below and if user enter 0 or negative number then print "Number 0 or negative can't be a prime number."

Example 1

Input

Enter a natural number(>0): 10

Output

10 is not a prime number.

Explanation

The given number is 10 which has more than 2 factors that's why it is not a prime number.

Input

Output

2 is a prime number.

Explanation

The given number is 2 which has only 2 factors and 2 is an only even prime number.

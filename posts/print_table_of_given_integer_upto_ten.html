Write a program in C programming language to print table of given integer up-to 10?

Table means simply multiples of a given number. Generally we study in school multiples or table of a number only for first 10 natural numbers but actually we can find infinite number of multiples of any given number.

Your job is to take a natural number to print the table or multiples from 1 to 10. Standard of output is shown below but if user enter 0 or negative integers then simply print "Your input is invalid. Please enter a natural number."

Example 1

Input

Enter a natural number to print the table(>=1): 5

Output

Table of 5: 5, 10, 15, 20, 25, 30, 35, 40, 45, 50.

Explanation

We have to print the table for first 10 natural number that's why here, we're getting first 10 mutiples of given natural number 5.

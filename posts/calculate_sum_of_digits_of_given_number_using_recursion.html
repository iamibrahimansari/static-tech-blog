Write a program in C programming language to calculate sum of digits of given number by using recursion?

Example 1

Input

Enter a whole number(>=0): 675

Output

Explanation

The given number is 675 which has 3 digits 6, 7 and 5. After add these digits we get 18 because 6 + 7 + 5 = 18.

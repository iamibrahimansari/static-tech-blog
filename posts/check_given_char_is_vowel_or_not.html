Write a program in C programming language to check whether given letter is vowel or consonant?

The English alphabet consists of 26 letters. Each letter has an uppercase ("capital letter") and a lowercase ("small letter") form. Five of the letters in the English Alphabet are vowels: A, E, I, O, U. The remaining 21 letters are consonants: B, C, D, F, G, H, J, K, L, M, N, P, Q, R, S, T, V, X, Z, and usually W and Y.

Your job is to take an english letter and check whether it is vowel or consonant and print the result as per the standard output which is shown below. If user enter other than english letter then print "Your input is invalid. Please enter only english letter."

Example 1

Input

Enter an english letter: E

Output

E is a vowel.

Explanation

In english alphabet there is 5 letters which are vowel. a/A, e/E, i/I, o/O and u/U are vowel.

Example 2

Input

Enter an english letter: z

Output

z is a consonant.

Explanation

In english alphabet all letters are consonant except a/A, e/E, i/I, o/O and u/U.

Input

Enter an english letter: /

Output

Your input is invalid. Please enter only english letter.

Explanation

/ is not an english letter. It is a special character.

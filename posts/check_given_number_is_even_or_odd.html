Write a program in C programming language to check whether a given number is even or odd?

Even numbers are those numbers that can be divided into two equal groups or pairs and are exactly divisible by 2. For example, 2, 4, 6, 8, 10, and so on. Those natural or counting numbers which are not even numbers are called odd numbers. For example: 1, 3, 5, 7, 9 and so on.

Your job is to take a natural number as input from user and check whether it is an even or odd as well as print the result as per the output standards which is shown below but if user enter 0 or negative integer then print "Your input is invalid. Please enter a natural number." on screen.

Example 1

Input

Enter a natural number(>=1): 5

Output

5 is an odd number.

Explanation

5 is not fully divided by 2 that's why it is an odd number.

Example 2

Input

Enter a natural number(>=1): 10

Output

10 is an even number.

Explanation

10 is fully divided by 2 that's why it is an even number.

Input

Enter a natural number(>=1): 0

Output

Explanation

No Explanation.

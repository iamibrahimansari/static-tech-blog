Write a program in C programming language to swap 2 given numbers by using pointers?

Swapping means change the value of two variables with each other. This is not a common swapping program, it is a tricky one because here you can't enter floating point number and if u do it then your input will be consider as wrong. So think a valid solution before code it. There will be no change in swapping code but careful when u take an input from user. This program must be using pointers to complete the swapping task otherwise your program will be consider wrong even though it will be produce right result. Here u can use temporary variable or not, it's up to you.

Your job is to take 2 integer numbers and swap them as well as print them on screen as per the outout standards which is shown beloe. If user enter a floating point number then print "Your input is invalid. Please enter integer numbers."

Example 1

Input

Enter 2 integer numbers: 5 10

Output

Before Swapping => First Number: 5 and Second Number: 10

After Swapping => First Number: 10 and Second Number: 5

Explanation

User entered 5 as First Number and 10 as Second Number. After swapping 10 becomes First Number and 5 becomes Second Number.
